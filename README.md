# Currency Converter - Конвертер валют

## Автор
**Имя и фамилия:** Агеев Никита

## Описание программы

Currency Converter - это приложение с графическим интерфейсом для конвертации валют в реальном времени. Программа использует внешний API для получения актуальных курсов валют, сохраняет историю конвертаций и позволяет экспортировать данные.

### Основные возможности:
- Конвертация валют по актуальным курсам из внешнего API
- Поддержка 160+ валют мира
- Возможность быстрой смены валют местами
- Сохранение полной истории конвертаций
- Экспорт истории в JSON формат
- Проверка корректности ввода данных

## Как получить API-ключ

### Вариант 1: Без ключа (используется в программе по умолчанию)
Программа использует бесплатный API [exchangerate-api.com](https://www.exchangerate-api.com), который работает без ключа до 1500 запросов в месяц. Этого достаточно для тестирования и личного использования.

### Вариант 2: С API-ключом (рекомендуется)
1. Перейдите на [exchangerate-api.com/sign-up](https://app.exchangerate-api.com/sign-up)
2. Зарегистрируйтесь (можно через Google или email)
3. Подтвердите email
4. Получите бесплатный API-ключ
5. В коде замените URL на: `https://v6.exchangerate-api.com/v6/YOUR_API_KEY/latest/`

### Вариант 3: Альтернативные API
- **OpenExchangeRates:** openexchangerates.org (требуется ключ)
- **CurrencyAPI:** currencyapi.com (100 запросов/месяц бесплатно)
- **Fixer:** fixer.io (требуется ключ)

Дополнительные библиотеки не требуются, используются только стандартные модули:

- `urllib` - для API запросов
- `datetime` - для временных меток
- `os` - для работы с файлами
